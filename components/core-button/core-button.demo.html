Example Usage of "core-button"

core-button color attribute

The core-button web component can include an attribute tag to control the color of the button. The color can be specified in the core-button tag as follows: color="primary". The possible color attribute values are primary, secondary, and dark. If no color attribute is specified a default color will be applied.

Primary
Secondary
Dark
